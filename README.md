# SamVisionAI

Real estate price prediction and analysis tooling inspired by Osama "Sam" Daloul. It combines ML models, engineered features, and local market data to provide valuation support for Canadian housing workflows.

## What This Is

SamVisionAI is a Streamlit application for:

- Uploading and parsing MLS PDFs
- Loading parsed sales data into PostgreSQL
- Building comparative market analysis views
- Estimating likely offer prices
- Comparing rental income scenarios

This is a focused internal tool, not a public SaaS product.

## Security Warning

Do not expose port `8501` publicly unless authentication is configured and tested. Docker Compose binds the Streamlit app to `127.0.0.1:8501` on the host so it stays local by default.

Never commit `.env` or private credentials.

`config.txt` is not used for database credentials. Keep any local config backups out of git because they may contain secrets.

## Docker Compose

Create a local `.env` from the safe example:

```bash
cp .env.example .env
```

Edit `.env` and replace all placeholders, then build and start the app:

```bash
docker compose up -d --build
```

Check the containers:

```bash
docker compose ps
docker compose logs -f samvision-app
```

Open the local app:

```text
http://127.0.0.1:8501
```

Stop the stack:

```bash
docker compose down
```

Quick local health check:

```bash
curl -I http://127.0.0.1:8501
docker logs samvision-app --tail=100
```

## Authentication Environment Variables

Required auth variables:

```bash
SAMVISION_ADMIN_USERNAME=admin
SAMVISION_ADMIN_PASSWORD_HASH=replace-with-generated-pbkdf2-hash
SAMVISION_SESSION_SECRET=replace-with-at-least-32-random-characters
```

`SAMVISION_ADMIN_PASSWORD_HASH` must be a PBKDF2-SHA256 hash generated by the helper script. `SAMVISION_SESSION_SECRET` should be a unique random value with at least 32 characters.

Generate a password hash interactively:

```bash
python scripts/generate_password_hash.py
```

Generate a session secret:

```bash
python -c "import secrets; print(secrets.token_urlsafe(48))"
```

For local testing only, the hash helper also accepts a command-line password:

```bash
python scripts/generate_password_hash.py --password "temporary-test-password"
```

## Database Environment Variables

Docker Compose also uses PostgreSQL settings from `.env`:

```bash
SAMVISION_DB_NAME=SamVision
SAMVISION_DB_USER=samvision
SAMVISION_DB_PASSWORD=replace-with-private-db-password
SAMVISION_DB_HOST=samvision-postgres
SAMVISION_DB_PORT=5432
```

The app reads these values at runtime. No database password is hardcoded in the repository.

## Local Streamlit Run

Install dependencies and export the required environment variables first:

```bash
pip install -r requirements.txt
export SAMVISION_ADMIN_USERNAME=admin
export SAMVISION_ADMIN_PASSWORD_HASH="your-generated-hash"
export SAMVISION_SESSION_SECRET="your-random-session-secret"
export SAMVISION_DB_NAME=SamVision
export SAMVISION_DB_USER=samvision
export SAMVISION_DB_PASSWORD="your-db-password"
export SAMVISION_DB_HOST=127.0.0.1
export SAMVISION_DB_PORT=5432
streamlit run app/streamlit_app.py
```

The checked-in Streamlit config binds local runs to `127.0.0.1:8501`.

## Later Public Access

Keep the app private until auth is enabled and verified. A later public setup can put Nginx Proxy Manager and Cloudflare in front of the app, but the upstream app should still be treated as an internal service and the proxy should target the local/private endpoint only.
